# Дипломный проект: Автоматизированное тестирование мобильного приложения "Мобильный хоспис"

## Описание проекта
Данный проект содержит документацию и автоматизированные тесты для мобильного приложения "Мобильный хоспис". Проект реализован с использованием стека Android Studio, Espresso/UI Automator.

* **[/docs](/docs):** Рабочая документация по проекту[cite: 1].
    * [План тестирования](/docs/Plan.md)[cite: 1].
    * [Реестр тест-кейсов и багов](/docs/Cases.xlsx)[cite: 1].

* **[Result.md](/Result.md):** Итоговый отчет о результатах автоматизированного тестирования.
* **[Test_Report.zip](/Test_Report.zip):** Архив с нативными отчетами о прогонах тестов (HTML).

## Технологический стек
* **Среда разработки:** Android Studio[cite: 1].
* **Фреймворки:** Espresso, UI Automator[cite: 1].
* **Отчетность:** Интегрирован Allure Framework; для локального анализа используются нативные HTML-отчеты Gradle[cite: 1].
