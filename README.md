# Дипломный проект: Автоматизированное тестирование мобильного приложения "Мобильный хоспис"

## Описание проекта
Данный проект содержит документацию и автоматизированные тесты для мобильного приложения "Мобильный хоспис". Проект реализован с использованием стека Android Studio, Espresso/UI Automator.

## Структура репозитория
* **[/fmh_android_15_03_24](/fmh_android_15_03_24):** Исходный код мобильного приложения и тестов.
* **[/docs](/docs):** Рабочая документация по проекту.
    * [План тестирования](/docs/Plan.md).
    * [Реестр тест-кейсов и багов](/docs/Cases.xlsx).
    * [Инструкция по запуску автотестов](/docs/README.md).
* **[Result.md](/Result.md):** Итоговый отчет о результатах автоматизированного тестирования.
* **[Test_Report.zip](/Test_Report.zip):** Архив с нативными отчетами о прогонах тестов (HTML).

## Технологический стек
* **Среда разработки:** Android Studio.
* **Фреймворки:** Espresso, UI Automator.
* **Отчетность:** Интегрирован Allure Framework; для локального анализа используются нативные HTML-отчеты Gradle.
