# Инструкция по запуску автотестов

## Технические требования
* **Среда разработки:** Android Studio версия Quail 1.
* **Эмулятор:** Pixel 4 (API 29 / Android 10.0).
* **Система сборки:** Gradle.

## Как запустить автотесты
Тесты запускаются через терминал из корневой папки проекта (`diplom/`) или из Android Studio.

### Через командную строку (рекомендуется):
Выполните следующую команду:
`./gradlew clean connectedAndroidTest`

*Примечание: команда `clean` перед запуском обязательна для удаления кэша предыдущих сборок.*

### Через Android Studio:
1. Откройте проект в Android Studio.
2. Перейдите в папку `app/src/androidTest/java/ru/iteco/fmhandroid`.
3. Нажмите правой кнопкой мыши на класс с тестами или на папку и выберите **Run 'Tests in...'**.

## Отчетность (Allure)
Результаты выполнения тестов доступны в двух форматах:
1. **Стандартный отчет Gradle:** После завершения тестов откройте файл `app/build/reports/androidTests/connected/index.html` в браузере.
2. **Allure Framework:** Настроена интеграция Allure. Для просмотра отчета (при наличии сгенерированных данных) используйте:
   `allure serve app/build/allure-results`
